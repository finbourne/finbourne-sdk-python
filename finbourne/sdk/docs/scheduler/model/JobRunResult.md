# JobRunResult

Represent a specific job's run
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **console_output_url** | **str** | Optional | The URL for the console output of the job run |
| **run_id** | **str** | Optional | Unique id of the job run |
| **job_id** | [ResourceId](ResourceId.md) | Optional | *No description available.* |
| **name** | **str** | Optional | Name of the job |
| **initialised_date** | **datetime** | Optional | The date when the job was initialised |
| **start_date** | **datetime** | Optional | The date when the job started running in Scheduler |
| **end_date** | **datetime** | Optional | End date of the job Defaults to null if the job has not started or not completed yet If value is null and job has not started, status will be &#x60;Pending&#x60; If value is null and job has not completed, but is running, status will be &#x60;Running&#x60; |
| **arguments** | **Dict[str, Optional[str]]** | Optional | All arguments used in in this job run |
| **environment_variables** | **Dict[str, Optional[str]]** | Optional | All environment variables used in this job run |
| **job_status** | **str** | Optional | Current job status |
| **job_result** | **str** | Optional | Description of the job result *(read-only)* |
| **description** | **str** | Optional | Description of the job |
| **schedule_id** | [ResourceId](ResourceId.md) | Optional | *No description available.* |
| **result_url** | **str** | Optional | URI to results. Defaults to null if not available |
| **manually_triggered_by** | **str** | Optional | UserId of the user that triggered the job. Defaults to null if job was ran on a Schedule |
| **message** | **str** | Optional | Output message generated by the job runner Value will be null when the job is in a &#x60;Running&#x60; or &#x60;Pending&#x60; status |
