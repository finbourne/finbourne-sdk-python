# ContractInitialisationEvent

Event to initialise a contract with a given limit against a LoanFacility.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **limit** | **float** | Required | Limit of this contract.  Must be positive. |
| **var_date** | **datetime** | Optional | Initialisation date of the contract. |
| **contract_details** | [ContractDetails](ContractDetails.md) | Required | *No description available.* |
| **instrument_event_type** | **str** | Required | The Type of Event.
