# BankruptcyEvent

A Bankruptcy (BRUP) event recording the legal status of a company unable to meet its financial  obligations. Pure informational marker — generates no transactions and has no position impact.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **effective_date** | **datetime** | Optional | Date of the bankruptcy filing or court ruling. |
| **notification_type** | **str** | Required | Notification type: NEWM (new announcement), REPL (replacement/correction), or CANC (proceedings dismissed). Available values: NEWM, REPL, CANC. |
| **claim_filing_deadline** | **datetime** | Optional | Date by which creditors must file a proof of claim. Optional — null when not applicable.  If provided, overrides EffectiveDate as the settle date of the resulting virtual transactions. |
| **narrative** | **str** | Optional | Free-text detail: court, jurisdiction, trustee, plan reference. Optional. |
| **instrument_event_type** | **str** | Required | The Type of Event.
