# CapletFloorletCashFlowEvent

Definition of a cap or floor (or collar) cash flow event.  This is an event that describes the occurence of a caplet or floorlet payment.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **entitlement_date** | **datetime** | Optional | Ex-Dividend / entitlement date of the cashflow payment.  Required field. |
| **payment_date** | **datetime** | Optional | Payment date of the cashflow payment  Required field. |
| **currency** | **str** | Required | Currency of the cashflow payment |
| **cash_flow_per_unit** | **float** | Required | The total payment per unit of the held instrument, if entitled.  Not required. |
| **instrument_event_type** | **str** | Required | The Type of Event.
