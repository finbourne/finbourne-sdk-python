# TermDepositPrincipalEvent

Definition of a Term Deposit Interest Event.  This is an event that describes the occurence of principal on a term deposit ().
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **currency** | **str** | Required | Currency of the principal payment. |
| **payment_date** | **datetime** | Optional | Payment date of the principal payment. |
| **principal_per_unit** | **float** | Optional | The principal payment made per unit of the held . |
