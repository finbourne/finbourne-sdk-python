# MaturityEvent

Definition of a Maturity Event  This is an event that describes the maturity of the instrument.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **maturity_date** | **datetime** | Optional | Maturity date of the instrument |
