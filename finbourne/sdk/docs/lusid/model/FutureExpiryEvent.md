# FutureExpiryEvent

Definition of a Future Expiry Event.  This is an event that describes the expiry of a Future instrument.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **expiry_date** | **datetime** | Optional | Expiry date of the Future instrument. |
| **settlement_currency** | **str** | Required | Settlement currency of the Future instrument. |
| **notional_amount_per_unit** | **float** | Optional | The notional amount of each unit in the Future instrument. |
| **instrument_event_type** | **str** | Required | The Type of Event.
