# ProtectionPayoutCashFlowEvent

Protection payout cashflow for credit default instruments (CDS or CDX).
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **ex_date** | **datetime** | Optional | The ex-dividend date of the cashflow. |
| **payment_date** | **datetime** | Optional | The payment date of the cashflow. |
| **currency** | **str** | Required | The currency in which the cashflow is paid. |
| **cash_flow_per_unit** | **float** | Optional | The cashflow amount received for each unit of the instrument held on the ex date. |
| **instrument_event_type** | **str** | Required | The Type of Event.
