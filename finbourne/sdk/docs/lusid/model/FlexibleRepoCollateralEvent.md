# FlexibleRepoCollateralEvent

Definition of FlexibleRepoCollateralEvent which represents a single collateral transfer as part of a repo contract  modelled as a FlexibleRepo, either as part of the purchase leg or repurchase leg, or any early closure.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **settlement_date** | **datetime** | Optional | Date at which the transfer of units settles. This is a required field. |
| **entitlement_date** | **datetime** | Optional | Date at which the recipient of the collateral is entitled to the units being transferred. This is a required field. |
| **amount** | **float** | Required | The total amount of collateral being transferred as part of the repo contract.  Signed to indicate direction of transfer. This is a required field. |
| **collateral_instrument** | [NewInstrument](NewInstrument.md) | Required | *No description available.* |
| **instrument_event_type** | **str** | Required | The Type of Event.
