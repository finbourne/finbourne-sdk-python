# UpdateDepositAmountEvent

Event to update the deposit be a given amount.
## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **var_date** | **datetime** | Optional | The date of the adjustment to the deposit. |
| **amount** | **float** | Required | The signed amount of the adjustment to make to the deposit. Positive implies an increase, and negative implies a decrease. |
| **instrument_event_type** | **str** | Required | The Type of Event.
